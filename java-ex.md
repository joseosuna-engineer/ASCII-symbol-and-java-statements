| Java expression | Statement |
|---|---|
| != | not equal |
| >= | greater than or equal to |
| <= | less than or equal to |
| ++ |Plus plus|
|--| Minus minus|
| /* |	Slash star|
| */ |	Star slash|
| -> |	Arrow|
| &&| 	And|
| \|\|  |	Or |
| .* | dot star|
